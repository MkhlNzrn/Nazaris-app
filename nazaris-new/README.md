# NAZARIS — сайт разработки для бизнеса

Сайт-визитка NAZARIS: разработка сайтов и приложений для бизнеса. Москва, Санкт-Петербург, Россия. Форма обратной связи отправляет заявки на nazaris@internet.ru.

## Запуск

```bash
cd nazaris-new
npm install
npm start
```

Сайт: http://localhost:3000

## Конфигурация

Скопируйте `.env.example` в `.env` и при необходимости измените:

- `EMAIL_USER` — почта отправителя (nazaris@internet.ru)
- `EMAIL_PASS` — пароль приложения Mail.ru
- `EMAIL_TO` — куда приходят заявки (nazaris@internet.ru)

## Деплой

1. Укажите ваш реальный домен (например nazaris.ru) в:
   - `index.html`: `canonical`, `og:url`, `twitter:*`, JSON-LD
   - `sitemap.xml`: тег `<loc>`
   - `robots.txt`: строка `Sitemap:`
2. Добавьте `icon/favicon.ico` для иконки вкладки.
3. Залейте проект на хостинг с Node.js и задайте переменные окружения (EMAIL_PASS и т.д.).

## Дополнительно для максимальной SEO

- **Домен**: после публикации замените все вхождения `https://nazaris.ru/` на ваш реальный URL.
- **Яндекс/Google**: добавьте сайт в [Яндекс.Вебмастер](https://webmaster.yandex.ru/) и [Google Search Console](https://search.google.com/search-console), при необходимости укажите мета-теги верификации в `<head>`.
- **Гео**: в коде уже есть упор на Москву, Санкт-Петербург и Россию (meta geo, JSON-LD, тексты). При появлении отдельной страницы для города (например «Услуги в Москве») добавьте её в `sitemap.xml` и сделайте ссылку с главной.
- **Ключевые запросы**: на странице уже используются формулировки вроде «разработка сайтов Москва», «приложения Санкт-Петербург», «лендинг под ключ», «автоматизация бизнеса». Для ещё большей видимости можно добавить блок «Частые вопросы» (FAQ) с целевыми запросами и схемой FAQPage в JSON-LD.

Если понадобятся отдельные страницы под регионы или список ключевых фраз под вашу тематику — напишите, подготовлю структуру и тексты.
